Projects

  • shopsClosed

    • Shop's Closed is a terminal based random string generator built using nodeJS. The example provided uses the context of a merchant in a video game telling the player the shop is not open, and to come back another day. shopsClosed works by generating a random number between 1-10, and based on which number is generated, selecting a pre-written section of the string. This occurs 3 times: once for the greeting, the name which the merchant calls the player, and the message telling the player to head home. You can see some example outputs in the image example.

    All things considered, shopsClosed was pretty easy to code, and is able to be used in a variety of scenarios; not just a text box in a video game. This project is where I learned how to use nodeJS, and git version control, and was my first locally built nodeJS application.

  • rockPaperScissors

    • This terminal based game of rock paper scissors was one of my first ever programs written in JavaScript. I actually did this one in the first learnJS course I took, and it helped spark a passion for learning JavaScript and Web Development. This program was built on the Codecademy integrated IDE using JavaScript and can be run via nodeJS.

    This program works by taking an input from the user via the IDE Code Editor, and comparing it to a randomly generated response of either rock, paper, or scissors. The winner is chosen via the traditional rules of Rock, Paper, Scissors. Currently the users selection is based on what is built in to the code, and can’t be changed externally. I plan to update this program in the near future, to enable user input via the terminal, rather than a coded-in selection of the user’s preferred throw.
